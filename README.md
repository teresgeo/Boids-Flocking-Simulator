# Boids Flocking Simulator

This project implements **Craig Reynoldsâ€™ Boids Algorithm (1986)** in MATLAB to simulate realistic flocking behavior using three simple emergent-behavior rules:

1. **Separation** - avoid crowding neighbors  
2. **Alignment** - steer toward the average heading of nearby boids  
3. **Cohesion** - move toward the center of mass of the flock
